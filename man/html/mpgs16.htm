mathcalc - interface to a TclLib calculator module

Synopsis

mathcalc

Description

 The mathcalc command provides a command line interface to a TclLib calculator module. The user is prompted (calc%) for input from standard input. The user's input is relayed to the calculator module and the results are displayed on standard output.

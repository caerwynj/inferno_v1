date

date - print the date

Synopsis

date

Description

 Prints to standard output the date, in the format

Tue Aug 16 17:03:52 CDT 1977 Current time is obtained from reading /dev/time. The conversion from Greenwich Mean Time to local time depends on the contents of /locale/timezone.

Files

/locale/timezone

/dev/time

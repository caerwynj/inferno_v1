netstat

netstat - rename/move files

Synopsis

bind '#I' /net
netstat

Description

 The netstat command prints information about TCP and UDP network connections. The information is obtained from the status file of each entry under the corresponding network directories, /dev/net/tcp and /dev/net/udp. The clone files are omitted from this report.

Files

/net/tcp/*

 /net/udp/*

 /prog/pid/status

Caveat

 This command requires a populated /net directory; therefore, there must be a prior binding of the system's #I driver to that directory.

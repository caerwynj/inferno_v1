error

error - return an error

Synopsis

Rerror tag [2] ename [64]

Description

 The Rerror request return an error string, of length 64 (ERRLEN in <lib9.h>), describing the failure of a transaction. It replaces the corresponding reply message that would accompany a successful call; its tag is that of the request.

There is no Terror in Inferno. Rerror is the only case in this protocol of a response message type without a corresponding request type.

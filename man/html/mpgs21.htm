os

os - interface OS commands (non-native Inferno only)

Synopsis

bind -ia '#C' /
os cmd...

Description

 On hosted implementations of Inferno, the os command allows the user to issue commands to the underlying operating system.

Use of the os requires the previous binding of #C, the command device.

Environmental Utilities

These environmental utilities are run from the host operating system and not from Inferno. They are as follows:.
asm

Dis assembler

emu

Inferno emulator

limbo

Limbo compiler

mk

 Maintain (make) related files. See the chapter on Using Mk in the Inferno Build Guide.
